# ⚡ ElectraPath AI: Civic Intelligence Dashboard

**ElectraPath AI** is a state-of-the-art civic intelligence platform designed for the 2026 Elections. It leverages a multi-stage Neural Audit Pipeline to ensure electoral compliance, candidate transparency, and voter empowerment.

![ElectraPath AI Banner](https://api.dicebear.com/7.x/shapes/svg?seed=ElectraPath&backgroundColor=6366f1)

## 🚀 Key Features

- **Neural MCC Analyzer**: Real-time screening of electoral content (transcripts, social media, speeches) against ECI Model Code of Conduct benchmarks using Google Gemini 1.5 Flash.
- **God-Mode Compliance Monitor**: A real-time neural feed for tracking campaign activities across multiple constituencies.
- **Interactive Candidate Intelligence**: Predictive scoring and detailed profiles for candidates in the Chennai Central constituency.
- **Civic Journey Mapping**: Personalized voter roadmap from registration to polling day.
- **Responsible AI**: Built in strict alignment with Google AI Safety and Fairness principles for civic technology.

## 🛠️ Technology Stack

- **Frontend**: React 19 + Vite 8
- **Intelligence**: Google Generative AI (Gemini 1.5 Flash)
- **Animations**: Framer Motion (Glassmorphism & Neural Transitions)
- **Icons**: Lucide React
- **Aesthetics**: Vanilla CSS (Tailored HSL Design System)

## 📦 Repository Size Optimization

- **Core Size**: < 3MB (Excluding `node_modules`)
- **Asset Strategy**: Scalable SVG iconography and dynamic API-driven media.
- **Performance**: High-fidelity HMR and optimized bundle splitting.

## 🚦 Getting Started

1. **Clone & Install**:
   ```bash
   git clone <repository-url>
   npm install
   ```

2. **Setup AI Key**:
   Create a `.env` file and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

3. **Launch Dashboard**:
   ```bash
   npm run dev
   ```
